Para crear un jar debemos ejecutar desde la terminal "mvn clean package"
Luego ir a la carpeta "target" y allí estará creado el .jar que vamos a copiar y pegar en la carpeta
donde tenemos el archivo dockerfile y docker-compose para que levante este .jar

En la carpeta que se encuentra en 
https://github.com/PabloDanielChiotti/dockerproject/tree/master/src/main/resources/docker-compose
podemos encontrar toda la información para ejecutar el docker-compose
